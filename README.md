# Rocket-Bird
Game from game engine

This game is build using the Game Engine based on Hazel Repo . It is 2D game much like flappy bird .

#Instructions to run the game
1. CLone the repo recursively
2. run the batch file is Script folder in root
3. Run the solution and play the game using Game Control Key SPACE BAR 

#Game Controls
Space Bar is the only key to use to play the game .
